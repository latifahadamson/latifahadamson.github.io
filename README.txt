How to edit:
1. Open index.html in a text editor (VS Code recommended).
2. To replace profile photo: overwrite images/profile.jpg with your photo (square recommended 600x600).
3. To update resume: replace resume.pdf with your CV (same filename).
4. To change links (LinkedIn/GitHub/email), edit the <a> href values in index.html.
5. To add projects: copy one of the <div class="project"> blocks in index.html and update image/title/description.
6. To edit style: assets/css/style.css

How to publish on GitHub Pages:
1. Create a new repository named: latifahadamson.github.io (your GitHub username followed by .github.io)
2. Upload all files from this folder into the repository root (use Upload files on GitHub or push via git).
3. Visit https://latifahadamson.github.io after a minute to view your site.
